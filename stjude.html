Data Science Tools

Learn more about how you can analyze pediatric cancer data

St. Jude Cloud

About St. Jude

The St. Jude PeCan Cloud provides various visualizations of pediatric cancer mutations. It pulls from multiple data sources from projects conducted at St. Jude and its collaborators.

Visit the Website

Protein Paint

ProteinPaint is a visualization tool that uses various graphs, charts, and apps that the user can adjust to explore various genomes and mutations that affect pediatric cancer.

Try it!

Genome Paint

GenomePaint is a large tool included inside ProteinPaint. Users can utilize it to analyze various pediatric tumor mutations. It also allows users to insert their own datasets for analysis.

Try it!

Visualization Community

St. Jude's Visualization Community is database containing pre-made visualizations from various sources. Users may choose to see ProteinPaint, GenomePaint, SJCharts, and Clinical Dictionary Browsers as visualization tools.

Try it!
