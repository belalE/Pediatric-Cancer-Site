Osteosarcoma Explorer

About the Osteosarcoma Explorer

The Osteosarcoma Explorer, created by UT Southwestern, is a database with information from a wide variety of studies involving osteosarcoma paitients. This tool uses user-friendly data analysis tools built inside the website that allow users to view the data themselves.

Visit the Website

Survival Analysis (Clinical)

This tool analyses the survival of paitients based on gender, data contributor, and ethnicity.

Survival Analysis (Expresssion)

This tool analyses how the survival of paitients is correlated to a certain type of gene that the user can choose.

Try it!

Survival Analysis (HER2 Protein Expression)

This tool analyses survival and its correlation to the prescense of the HER2 protein in the paitient. Users may choose to analyze overall survival or event-free survival.

Try it!

Tutorial

Survival Analysis (CNV)

This tool analyses the copy number variations (CNVs), or the loss or gain of DNA segments in paitients and its correlation to a certain gene. Users can select which gene to analyze.

Try it!

Tutorial

Survival Analysis (ctDNA)

This tool analyses survival and how it correlates to the prescence of ctDNA in the paitient's bloodstream. Users may choose to analyze overall survival or event-free survival.
